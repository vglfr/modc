## modc - Language Which Compiles ASM in Modules

+ migrate from numc
+ spool Prog's

- compile Prog's

- conditionals (unconditional)
- comparisons
- boolean types
- conditionals (conditional)
- first order functions
- fX, iX, uX types

Conventions:
- arguments are |'0 '1 ..|
- constants are |?0 ?1 ..|
