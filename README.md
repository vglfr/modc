## modc - Language Which Compiles ASM in Modules

+ migrate from numc
+ spool Prog's
+ compile Prog's

- partial functions (autocurry)

- conditionals + comparisons
- types (boolean + double)
- first order functions (boolean + double + function)

- user-defined data types - haskell
- module system - ocaml
- x86 / ARM backend - c++

Conventions:
- arguments are |'0 '1 ..|
- constants are |?0 ?1 ..|
